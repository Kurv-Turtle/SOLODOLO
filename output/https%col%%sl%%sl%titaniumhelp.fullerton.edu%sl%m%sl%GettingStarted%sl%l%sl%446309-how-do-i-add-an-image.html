How do I add an image?

You will learn how to add an image to your course in TITANium.

Do not try to copy/paste an image from another website. You should use images that you have permission to use, are in the public domain, or are under Creative Commons license.

First, ask yourself where you want to put the image:

In a textbox (in almost any Activity follows the same process)

Follow these steps to add a course banner to be placed at the top of your course main page.

1. Enter your course in TITANium.

2. Click on the Gear icon.

3. Turn editing on.

The Gear menu reveals the following links: Edit settings, Turn editing on, Filters, Gradebook setup, Outcomes, Backup, Restore, Import, Recycle bin, Reset, and More...

4. Click on Edit and select Edit section.

5. Drag the image to the textbox.

Alert: The image file must be located on your computer as a file. It cannot be in your email, on a website, or in the Cloud. If it is in your email or a website, you must download it to your computer before you can upload it to your course.

6. Double-click the image in the textbox.

Or, you can click once on the image and then click on the Insert image icon in the textbox toolbar.

7. Deselect the 'Description not necessary' checkbox.

8. Type a description of the image.

The text description should be meaningful for a student who is using a screen reader in your course.

9. Click on Save image

10. Click on Save changes

11. The image now displays on the course main page in the Topic description.

In a Label on the course main page

Follow these steps to add an image to a label.

1. Enter your course in TITANium.

2. Click on the Gear icon.

3. Turn editing on.

The Gear menu reveals the following links: Edit settings, Turn editing on, Filters, Gradebook setup, Outcomes, Backup, Restore, Import, Recycle bin, Reset, and More...

4. Click on Add activity or resource.

5. Select Label.

6. Click on Add.

7. If you add text in the Label, click on the spot where you want the image to appear.

8. Drag the image file to the textbox.

You can drag and drop an image file to almost any open textbox within TITANium.

9. Double-click on the image.

10. Type a description.

11. Select Alignment.

12. Click on Save image.

13. The image now displays in your textbox.

14. Click on Save and return to course.

15. The image now displays on your course main page within the Label.

•As a file upload on the course main page

Follow these steps to add an image file to your course main page.

1. Enter your course in TITANium.

2. Click on the Gear icon.

3. Turn editing on.

The Gear menu reveals the following links: Edit settings, Turn editing on, Filters, Gradebook setup, Outcomes, Backup, Restore, Import, Recycle bin, Reset, and More...

4. Click on Add activity or resource.

5. Select File.

6. Click on Add.

7. Type a title.

8. Drag the file to the upload area.

9. Select your preference for how the student should view the image file.

The Appearance choices are as follows.

  • Force download (default) will directly download the file to your downloads location on your computer.
  • Embed will display the image on the page with the TITANium blocks and header.
  • In frame will display the image with only the TITANium header.
  • New window will open a new window or tab and display only the image.
  • Open will display the image only. You will need to click on the web browser's back button to return to your course.
  • In pop-up will display the image in a new pop up window. Web browser pop-up blockers will typically block this the first time a user clicks on it.

For this example, Embed has been chosen.

10. Click on Save and return to course.

11. The icon and title for the image appears on the course main page.

12. Clicking on the link will display the image embedded in the web page.

As a file upload in a Folder.

Follow these steps to upload image files to a Folder. Read this guide to learn how to add a Folder.

1. Enter your course and click on the Folder in your course

2. Click on Edit

3. Drag the file or files to the upload area.

4. Wait until the icon appear, then click on Save changes.

5. The image files are now in your Folder.
