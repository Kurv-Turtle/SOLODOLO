On the Horizon

The Campus Web Solutions team is continuously working on improvements to the Drupal offering. The following list is a guide to what is "On the Horizon". Please note that release dates are tentative as new projects and other issues arise. The team has to fully test new modules and changes to make sure everything works as intended. This can be an extensive, lengthy process. Drupal sprint releases are scheduled once per month, typically on the 1st or 2nd Saturday of the month. 

Items last updated: February 2022

Item Estimated Release Type Teams Description
Accessibility Fixes Ongoing global change UComm (color guidance) Campus Web Solutions Fix color contrast issues, labels, aria roles, everything specific to the template
SEO updates Ongoing global change UComm & Campus Web Solutions Any SEO changes that need to be done on the template level
Bubble Grid March 2022 enhancement Campus Web Solutions Larger text display, do not uppercase, positioning centered
Info Card Grid March 2022 enhancement Campus Web Solutions Allow option for left or centered. Allow for bullets to display.
Boxes Spring 2022 enhancement UComm (design) & Campus Web Solutions Div box styles that affect specific text in the WYSIWYG editor
Aggregate Feeds Spring 2022 global change Campus Web Solutions Internal solutions for aggregating and display of feeds. 
Font Awesome Icons Summer 2022 global change Campus Web Solutions Repository of icons available in the editor & applicable fields
Search field on site results Summer 2022 global change Campus Web Solutions Add field to update search keywords while on search results page
Layout: In place editing PENDING enhancement Drupal Community Use of "pencil" icons in view mode to edit blocks
Layout: Inline vs Global PENDING enhancement Drupal Community Ability to use the Layout Builder tool to create block and select whether they are inline vs. global at the time of creation
Lazy Load evaluating module Campus Web Solutions Lazy Load images
Emergency Notification System Integration PENDING external Campus Web Solutions, UComm, OEM ability to automatically integrate ENS messages across all Drupal websites
Events Content Type re-haul PENDING enhancement Campus Web Solutions, UComm Reevaluate the Events content type to be more dynamic and have other options (such as calendar view)
