Browser Requirements

  • System Requirements: Windows 7 or higher, Mac OS X 10.6 or higher
  • Supported desktop browsers: Google Chrome or Mozilla Firefox, Safari 8 or higher, **Internet Explorer 11, Microsoft Edge
  • ​ Supported mobile browsers: Chrome, Firefox, Safari (iOS), Android native brows​er (Samsung devices only)
  • ​ Java: http://www.java.com/en/download/
  • Internet Connection: 5-10mbps download speed is recommended. Check your connection at www.speedtest.net.
  • Speakers/headphones​

**Please see Microsoft's statement regarding Internet Explorer support.

Captioning Requirements

Enable Java in browser for the captioning to work. Turn off popup blockers for broadcast.

Java Version 6 update 12 or later
(build 1.6.0_12-b04) is required on your computer for captioning. Use the Software Update feature (available on the Apple menu) to check that you have the most up-to-date version of Java for your Mac.​
