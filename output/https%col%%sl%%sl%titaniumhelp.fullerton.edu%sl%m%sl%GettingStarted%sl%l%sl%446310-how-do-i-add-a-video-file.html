How do I add a video file?

You will learn how to add a video file to your course in TITANium. There are several different ways to add a video file.

The video file must be less than 100 MB. If you have a file larger than 100 MB, consider using Dropbox, ShareStream or YouTube.

First, decide where you want the video to be and how it should be presented to the students:

•As a file on the course main page

Follow these steps to upload a video file to your course main page.

1. Enter your course in TITANium.

2. Click on the Gear icon.

3. Click on Turn editing on.

The Gear menu reveals the following links: Edit settings, Turn editing on, Filters, Gradebook setup, Outcomes, Backup, Restore, Import, Recycle bin, Reset, and More...

4. Drag the video file from your computer to the desired Topic section.

In this example we are uploading a video file, but you can upload almost any type of file this way.

5. The link to the video file displays on the course main page.

By default, when you upload a file by dragging it to the course main page, when you click on the link the file will download.

To change the action that occurs when you click on the link to a video file, click on Edit and then select a different setting for Appearance. You can select from Force download, Embed, In frame, Open, New window, or Open in pop-up.

•As a video to play in a textbox (like in a Label, Page, Book, etc.)

Follow these steps to upload a video file to a Page.

1. Enter your course in TITANium.

2. Click on the Gear icon.

3. Click on Turn editing on.

4. Click on Add an activity or resource.

5. Select Page and click on Add.

6. Type a name for the link on the course main page.

7. Type a description if desired.

The Description field is optional and can be displayed on the course main page if desired.

8. Type text in the Page content textbox.

9. Click on Manage files in the menu toolbar.

If only uploading one video file, you may want to click on the Insert media button instead. However, since in this example we are uploading several videos, the File manager is a quicker choice.

10. Drag the files to the upload area.

Instead of dragging the file(s), you can click on the Add (looks like a sheet of paper with a plus sign on it) button to navigate to where your files are.

11. Wait until all of the icons appear, then click on the X to exit.

Note that the current maximum upload limit per file is 100MB. If you have a larger file, then you need to use a different method of providing your videos to your students, such as ShareStream, YouTube or DropBox.

12. Place the cursor where you want to insert the video file.

13. Click on the Insert media button in the toolbar menu.

14. Click on Browse repositories.

15. Click on the title of the desired video file.

16. Click on Select this file.

17. Click on Insert media.

At this point, you can click on the Video tab to add a captions file (VTT) or subtitle file (VTT) and set the display dimensions if you want them to be different than the actual video dimensions.

18. The video file is now listed in the textbox.

19. Follow the same procedure for the remaining videos.

20. Click on Leave.

Once you upload some files using the Manage files button, you will likely see this message when you navigate from this page.

21. Click on Save and return to course.

22. The Page link now displays on the course main page.

23. Clicking on the link with display the Page with the text and videos.

24. Click anywhere on the image to play the video.

•As a file in a Folder

Follow these steps to add a video file to a Folder.

1. Enter your course and click on the link to the desired Folder.

Read this guide to learn how to add a Folder to your course.

2. Click on Edit.

3. Drag your files to the upload area.

In this example we are uploading a video file, but you can upload almost any type of file this way.

4. Wait for the video icons to appear, then click on Save changes.

The icon for each file type will appear once the file has been uploaded.

5. Your files are now listed in the Folder.

Files listed in a Folder will download when clicked; you cannot change that setting.

•As a video in a Quiz

Follow these steps to upload a video to a Description question for a Quiz in the course question bank.

1. Within your course, click on the Gear icon.

2. Click on More.

The Gear menu reveals the following links: Edit settings, Turn editing on, Filters, Gradebook setup, Outcomes, Backup, Restore, Import, Recycle bin, Reset, and More...

3. Scroll down to the Question bank section.

4. Click on Categories.

The reason to create a new Category is to organize your questions for later use in Quizzes. However, it is not a requirement.

5. Scroll down to the Add category section.

6. After typing a name, click on Add category.

7. Click on the (0) to the right of the name category name.

The number within parentheses () is the number of questions within that category.

8. Click on Create a new question.

9. Select Description and click on Add.

The Description question type is just for information. There is no actual question. It is good for placing a reading passage, images, or video where several questions will refer to the same piece of information.

10. Type a name for the question.

11. Type some text in the Question text field.

12. Position the cursor where you want to insert the video.

13. Click on the Insert media button in the toolbar menu.

14. Click on Browse repositories.

15. Click on Upload a file.

For this example, we are uploading a video from our local computer; However, you can also link to a DropBox video file, a YouTube video, or an Embedded file.

16. Click on Choose File.

17. Navigate to your file.

18. Double-click on the desired file.

19. Select the desired license.

20. Click on Upload this file.

21. Click on Insert media.

At this point, you can click on the Video tab to add a captions file (VTT) or subtitle file (VTT) and set the display dimensions if you want them to be different than the actual video dimensions.

22. The video link is now in the textbox.

23. Scroll down the page and click on Save changes.

24. The video is now ready to be linked to a quiz in your course.

Read this guide to learn how to create a Quiz and link questions.
