How do I upload a SoftChalk lesson to my course in Canvas?

You will learn how to upload a zip package of a lesson, such a SoftChalk lesson, Captivate, or Camtasia to your course in Canvas.

A zipped SoftChalk lesson will be used in this example.

Keep in mind this zip package is for a student to interact with the instructional content but will not report a score to the gradebook.

1. Enter your course.

2. Click on the Files navigation item.

3. Click on +Folder.

It is not necessary to add a folder, it is being done for file management. If you plan to upload more than one zip package, it can be helpful to have them organized by folders.

4. Type a name for  the new folder.

5. Click on the Check mark.

6. Click on the folder.

7. Drag the zip folder to the upload area.

8. Click on Expand It button.

9. Allow the zip to upload and unpack.

There are two ways to link to this lesson.

I would like to insert a link within a textbox.

1. Locate the entry file for your lesson.

The starting page is different for different programs. For SoftChalk Create it is "index.html". For Camtasia, it is the name of the project .html.

2. Right mouse click and select Copy Link.

3. Click on Modules.

4. Click on +.

5. Select the type of item you want to create.

You can place this type of content within any Rich Content Editor (textbox). In this example, Discussion has been chosen.

6. Select [New Topic].

7. Type a name for this discussion.

8. Click on Add Item.

9. Click on the discussion link.

10. Click on Edit.

11. Add text, images, etc to the textbox.

12. Select some text or an image.

You can use text or an image to link to your lesson.

13. Click on the Links icon.

14. Select External Links.

15. Paste the link you had previously copied.

16. Click on on Done.

17. Save.

18. Click on the link.

19. The lesson displays in a new tab.

20. When the discussion is all set up, click on Publish.

21. Click to publish the module.

Article Summary

You have now learned how to add a zip packaged lesson to a textbox.

I would like to create a direct link within a Module.

1. Click on Modules.

2. Click on +.

3. Click on the dropdown menu to select File.

4. Scroll down to the index.html file.

5. Click on Add Item.

6. Click on the three dots menu item.

7. Select Edit.

8. Type a new title.

9. Click on Update.

10. Click on the link to verify that it works.

11. The lesson opens in the same window.

Article Summary

You have now learned how to link to a zip-packaged lesson embedded in the same window.
