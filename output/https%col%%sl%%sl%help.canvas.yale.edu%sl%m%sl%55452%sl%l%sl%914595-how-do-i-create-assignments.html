Creating an Assignment

Instructors can use Assignments to create and track graded and ungraded assignments, including online and offline submission, no submission, discussions, and quizzes.

Table of Contents

This article will address the following information:

  • Creating the Assignment
    • Assignment Settings

Creating the Assignment

  1. Click on "Assignments" from the course navigation.
  2. Click "+Assignment" at the top-right corner.
  1. Name your assignment in the top box.
  2. Enter a description or assignment details in the Rich Content Editor.
  3. If you would like to attach files, you can select the file from the Documents icon.
    1. For more information on using the Rich Content Editor, review the "Using the Rich Content Editor" article.

Assignment Settings

Below the Rich Content Editor are assignment settings that you can selected to fit your assignment's specific needs and configuration.

  1. Points - Set the number of points the assignment is worth. If ungraded, enter "0."
  2. Assignment Group - Select an assignment group if any have been created.
    1. See the article "Creating Weighted Assignment Groups" for more information.
  3. Display Grade As - Select the type of grade that shows up in the gradebook and on the students' view.
    1. Percentage
    2. Complete/Incomplete
    3. Points
    4. Letter Grade - Uses the grading scheme
    5. GPA Scale - Uses the grading scheme
    6. Not Graded - If the assignment has no submission AND is ungraded

Check off the "Do not count this assignment towards the final grade" option below the "Display Grade As" drop down if the assignment has a submission, but should not be counted towards the course's final grade.

  1. Submission Type and Plagiarism Review - Set the submission type. If a online file is selected, you can also use the plagiarism review tool Turnitin.
    1. No Submission
    2. Online
    3. On Paper
    4. Text Entry
    5. Website URL
    6. Media Recording
    7. File Uploads
    8. External Tool
  2. Group Assignment - Select to designate the assignment as a group assignment.
  3. Peer Reviews - Select to have students review their peers' work to provide feedback.
  4. Assign - Select who and when will see the assignment, as well as set the due date.
  5. Click "Save" or "Save & Publish" when finished.

For more help, please contact canvas@yale.edu.
