How do I edit, delete, or hide a grade item?

After you have created some Grade items for your gradebook, you may need to edit, delete, or hide those grade items. You will learn how to edit, delete, or hide a Grade item in your course in TITANium.

This guide has been updated for TITANium 2019-2020.

1. Enter your course in TITANium.

2. Click on the Gear icon.

3. Select Gradebook setup.

4. The Gradebook setup page will display.

What would you now like to learn to do?

I want to edit a Grade item.

Follow these steps to mark a grade item for extra credit.

1. Locate the item you want to change to extra credit.

In this course, the course aggregation method is set to the default which is Natural - summing the grades.

2. Click on Edit.

3. Select Edit settings.

4. Check the Extra credit box.

Scroll down the Edit grade item page to the Extra credit check box. Click on the check box.

5. Click on Save changes.

6. Note the + sign with the dots under it.

The + sign with the dots under it signifies that this grade item is set as Extra credit.

7. Note that the course total is now 10 points less.

The course total in the Natural aggregation method will automatically adjust the course total to account for the grade items marked as Extra credit. Since the item marked is worth 10 points, the course total is now 10 points less than it was.

You have now learned how to edit a grade item.

I want to delete a Grade item.

Follow these steps to delete a grade item.

1. Locate the grade item.

2. Click on Edit.

3. Select Delete.

Wait! It doesn't say "Delete".

If you are trying to delete an Activity, you cannot delete it from within the Gradebook. You must either set the grade value in the activity to "None" which will remove it from the gradebook, or you must delete it from the course main page which will delete it from the gradebook.

In this sample course, the quiz does not allow you to delete it from the gradebook because it is an activity.

1. Return to your course main page.

2. Click on the Gear icon.

3. Select Turn editing on.

4. Scroll down to the activity.

5. Click on Edit.

6. Select Delete.

4. Click on Yes.

There is no "Undo" for this kind of action.

5. That grade item is now gone.

The activity has now been deleted.

I want to hide a Grade item.

Follow these steps to hide a grade item that is currently visible to students.

1. Click on Edit.

2. Select Hide.

3. That grade item is now hidden.

The light grey text indicates that the grade item is hidden from students.

4. That grade item is now hidden from students in the User report.

The User report is the view of the gradebook that the students have. Note that the desired grade item is not visible.

I want to show a hidden Grade item.

Follow these steps to set a hidden grade item to be visible to your students.

1. Click on Edit.

In this sample, the Bio and Photo grade item is currently hidden in the gradebook. Students cannot see it and by default, it does not count toward the course total (in student view).

2. Select Show.

Wait! I cannot select Show.

If the only choice is Edit settings, then you are trying to edit an activity, not a plain grade item. To Show or Hide an activity, go to the course main page.

3. The Grade item is now visible to students.

Article Summary

You have now learned how to edit, delete, hide, or show a Grade item in the gradebook in your course in TITANium.
