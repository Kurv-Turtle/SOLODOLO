Tips for eCourse Completion
  1. Make sure you are using a supported web browser and operating system.
  2. Either disable pop-up blockers or create an exception so that the browser will allow pop-ups from https://uc.sumtotal.host.
  3. Do not leave eCourses paused for extended periods of time. If you need to step away, exit the course until you can return. Leaving a course paused for too long may prevent it from reporting progress back to the system server. Any disruption in internet connectivity while attempting a course can prevent progress from being recorded.
  4. Exit a course using the Exit link from the course menu when available. If needed, wait for the course to indicate it is okay to close the browser window before doing so. Closing a course window too soon may prevent it from reporting progress back to the system server.
  5. Save a screenshot of your quiz results and/or completion confirmation screen.
  6. Enable & Clear Cookies: You can view this setting in the browser’s security dialog box. Clearing the browser's cookies can often resolve unexpected performance issues. Note, some privacy add-ons block cookies. If you have installed any add-ons, plug-ins, or toolbars in your browser, especially those designed to protect privacy by blocking cookies, these may interfere with the system's ability to record your progress and/or eCourse functionality. You may want to disable them while attempting to complete online courses or use a browser without such.
  7. Check Available Browser Support: For help troubleshooting browser settings mentioned above, and more, refer to your browser's support site (Chrome, Edge, Firefox, Safari).
Browser Compatibility

SUPPORTED BROWSERS
The UC Learning Center generally supports the most CURRENT and the previous TWO versions of the browsers listed below.

 Recommended Settings by Browser Recommended Settings for All Browsers
Chrome

Under Settings > Advanced > Privacy and Security > Content Settings:

  1. Images: Show all images
  2. JavaScript: Allow all sites to run JavaScript (recommended)
  3. Plug-ins: Run automatically (recommended)
  4. Pop-ups: Allow all sites to show pop-ups
  5. Unsandboxed plug-in access: Ask me when a site wants to use a plug-in to access my computer (recommended)
  • Pop-Ups allowed
  • Cookies enabled
  • JavaScript enabled
  • Java may be required for certain online courses
Firefox
  1. Under Options > Content:
    • Uncheck Block Pop-Up windows
    • Check Accept cookies from sites
  2. Under the Security tab — check the Warn me when sites try to install add-ons
Microsoft Edge
Ensure you enable cookies, allow pop-ups, and do not run in compatibility mode. Edge does not require any unique setting configurations
Safari

We currently advise Mac users use Firefox or Chrome. As ITS has noted an increase of issues with Safari particularly around form buttons, rendering, interactive elements etc.

Under Options > Preferences configure the following tabs:

  1. Web Content:
    • Check Enable plug-ins
    • Check Enable Java
    • Check Enable JavaScript
    • Uncheck Block pop-up windows
  2. Check: Ask before ending a non-secure from to a secure website
  3. Privacy:
    • Under Block cookies: select From third parties and advertisers
    • Under Limit websites access to location services: select Prompt for each Website Once

Additional Device Requirements:

  • Devices must have a screen resolution of at least 1024x768.
  • Tablets are strictly supported in landscape mode.
     
Mobile Application & Device Compatibility

Mobile phone support is restricted to the SumTotal Mobile App only. Mobile web browsers are not supported. For a demonstration of accessing and using the mobile app, see the Learner Guide. The mobile site URL is https://uc.sumtotal.host. 

  • iPhone or iPad* (iOS 14 or 15 or iPadOS 14x or 15.x)
  • Android phones or tablets* (9.x, 10.x, 11.x, or 12.x)
  • Devices running Windows 10.x

Some tablets are strictly supported at 1024x768 resolution and only in landscape mode.

NOTE: There are limited eCourses available via the mobile app. The app has limited functionality, e.g., registration to in-person courses is not available. We recommend using a desktop or laptop, or a tablet running Windows 10, to access the UC Learning Center.
