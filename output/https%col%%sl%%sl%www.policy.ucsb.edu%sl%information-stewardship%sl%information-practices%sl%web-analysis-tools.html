Web Analysis Tools

The UCSB web site, www.ucsb.edu and associated pages, collect information to improve functionality and content and to monitor performance. Data is used to help answer specific questions about the usage and performance of the web site or individual web pages. UCSB uses third party web analysis services to help gather and analyze this information. UCSB’s Privacy Notification Statement can be found at: http://www.policy.ucsb.edu/privacy-notification.

Use of Web Analysis Tools

The UCSB web site, www.ucsb.edu and associated pages, use web analysis tools, including but not limited to Google Analytics, a web analytics service provided by Google, Inc. (“Google”). Web analysis tools use “cookies”, which are text files placed on your computer to help analyze how users use this Web page. The information generated by the cookie about your use (including your IP address) will be transmitted to and stored by analytics provider. The analytics provider will use this information for the purpose of evaluating your use of this page, compiling reports on activity for UCSB staff and providing other services relating to Web page activity and internet usage.

The analytics provider may also transfer this information to third parties where required to do so by law, or where such third parties process the information. You may refuse the use of cookies by selecting the appropriate settings in your browser. By using these Web pages and accepting cookies, you consent to the processing of data about you in the manner and for the purposes set out above.
