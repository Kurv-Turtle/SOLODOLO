About Stanford Accounts

The Stanford Accounts Application (called Accounts, for short) is the central web application for the creation and maintenance of your SUNet ID, the primary account for all students, faculty, and staff.

If you are a new employee or student, you are likely here to create a SUNet ID so you can access Stanford services. Please be sure to have your University ID on hand, proceed back to the main page, and select Create. Documentation on that page will help you find your University ID.

If you have forgotten your password, you can select Reset Password on the main page to pick a new one. If you know your password and want to pick a new one, or want to make other changes to your account, please select Manage on the main page.
