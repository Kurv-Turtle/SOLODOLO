How do I export a test or items from Respondus to a file?

You will learn how to publish a question bank or test which you have created in Respondus (or imported into Respondus) as a single (zip) file. This guide will show you how to use two of the most popular file types: Blackboard and QTI.

If you are not publishing directly to Canvas using the Canvas personality, you can export your test items to file as an intermediate step. Then import that file into your LMS.

Canvas recognizes several formats for file upload of test items; however, Classic Quizzes and New Quizzes require different file types, so we are demonstrating Blackboard zip file for Classic Quizzes and QTI for New Quizzes.

Decide Classic Quizzes or New Quizzes:

Exporting to Blackboard Export ZIP Package

Creating a Blackboard export ZIP package allows the creation of a Classic Quiz test item bank.

1. Open Respondus and change the Current Personality to “Blackboard.”

2. Open, create or import a Respondus test or question bank.

In this sample, we already have a Respondus question bank we have previously imported, but you can also create questions from scratch, use the Test Bank Network or the Exam Wizard to get started.

3. Select the desired test bank or exam.

4. Click on Open.

5. Click Preview + Publish.

6. Select Publish Wizard.

7. Select "Save pool to local file for manual uploading".

8. Click Next.

9. Click Save As…

10. Type a file name with no spaces.

11. Click on Save.

12. Click on OK.

13. Select Test with points.

If you are going to upload this question bank to Canvas, you will need to select to add points to the questions; otherwise, you will have to edit every single question in Canvas to add a point to it or add them to a quiz within a group assigning points as you create the group. (Unless you want to build a quiz with no points.)

14. Click on Finish.

15. Click on OK.

You have now created a ZIP file which you can use to import into Canvas for use with Classic Quizzes.

Exporting to QTI ZIP Package

The QTI XML ZIP package format can be used to create either a Classic Quiz test item bank or a New Quizzes test item bank in Canvas.

1. Current Personality set to “IMS QTI”.

2. Open, create or import a Respondus test or question bank.

In this sample, we already have a Respondus question bank we have previously imported, but you can also create questions from scratch, use the Test Bank Network or the Exam Wizard to get started.

3. Select the desired test bank or exam.

4. Click on Open.

5. Click Preview + Publish.

6. Click on Save QTI XML File.

7. It is recommended to leave the default QTI settings.

The defaults are to use Points as percentages 0-100% and QTI XML zip file using IMS Content Packaging 1.13.

8. Save As.

9. Type a name for the file without spaces.

10. Click on Save.

11. Click on OK.

Article Summary

You have now learned how to save your Respondus test bank or exam as a file for upload to your course in Canvas.
